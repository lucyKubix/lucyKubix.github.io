## <a href="https://www.asmodee.co.uk/" target="_blank">Asmodee</a>
### <a href="https://github.com/kubixmedia/asmodee-cli" target="_blank">Repo</a>

* ### Custom element architecture:
Modular ES6 web components communicating via shared variant-change and custom events, keeping complex interactions decoupled and maintainable at scale.
* ### Product comparison system:
localStorage-powered product comparison with Section Rendering API support, available as both a drawer and modal with dynamically fetched product data.
* ### Quick Add drawer:
AJAX-driven product drawer with full variant support, hover-based prefetching for faster interactions, and safe request cancellation to prevent race conditions.
* ### Variant-aware media galleries:
Swiper-based galleries that re-filter and reinitialise when variants change, supporting slider and stacked layouts with optional media grouping by variant option.
* ### Dynamic metafield system:
Flexible product metafield rendering with conditional visibility, logged-in-only rules, and defensive handling for missing or incomplete data.
* ### SparkLayer B2B pricing:
Custom element integration with SparkLayer enabling dynamic B2B pricing, VAT calculations, price breaks, and customer-specific pricing via the SparkLayer API.
* ### Internationalization support:
RTL language detection with responsive country and language localization forms that adapt between header and mobile navigation.
* ### Store locator:
Dual integration with Storemapper and Stockist.co widgets, allowing flexible store finder implementations across multiple page types.
