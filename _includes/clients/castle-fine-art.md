## <a href="https://www.castlefineart.com" target="_blank">Castle Fine Art</a>
### <a href="https://github.com/kubixmedia/castle-fine-art" target="_blank">Repo</a>

* ### Nosto Integration:
Nosto placement blocks are used across product, collection, homepage, search, cart, and 404 pages. The cart drawer includes cross-sell recommendations, and cart updates are tracked for personalization. A main-collection-no-nosto section allows excluding Nosto on specific collections.
* ### Artist Hub System:
Artist metaobjects power dynamic artist showcases with featured works pulled from metafields, supporting multiple featured collections with intelligent fallback logic to ensure consistent 4-image displays across artist profiles.
* ### Event Management Architecture:
Comprehensive event system built on metaobjects with location-specific drawers, embedded registration forms via Globo Formbuilder, dynamic media galleries (video/image), and date-based filtering for upcoming/past events with automatic status detection.
* ### Custom Finance Drawer:
Advanced V12 finance calculator with interactive deposit slider, real-time payment plan calculations, Klarna integration, and expandable option cards. Features dynamic APR calculations, loan amount validation, and responsive modal/drawer presentation with market-specific finance product availability.
* ### Recently Viewed Products Drawer:
Sophisticated recently viewed drawer system with auto-open on homepage return, session-based visibility controls, custom scrollbar with hover states, and configurable display rules (once per page type, days before showing again). Includes header icon integration and localStorage/sessionStorage tracking for optimal UX.
