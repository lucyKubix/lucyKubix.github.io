## <a href="https://www.tedbaker.com" target="_blank">Ted Baker</a>
### <a href="https://github.com/kubixmedia/ted-baker" target="_blank">Repo</a>

* ### Metafield colour swatches:
Colour selection built from sibling product metafields, parsing colours from product titles and generating solid or gradient swatches via theme settings.
* ### Gift card handling:
All gift card product links redirect to the Givy app to avoid regional and app URL issues.
* ### Web component architecture:
50+ ES6 custom elements communicate via shared variant change events, keeping the system modular and maintainable.
* ### International SEO:
Automatic hreflang output and smart URL rewriting for UK, EU, and US markets.
* ### Quick Add drawer:
AJAX-loaded product drawer with full variant support, accessibility safeguards, and safe request cancellation.
* ### Variant-aware galleries:
Swiper-based galleries that re-filter and reinitialise per variant, supporting multiple layout modes.
